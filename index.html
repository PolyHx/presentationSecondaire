Boucles

Une boucle while est un morceau de code qui sera répeté tant qu'une condition sera vraie.


						int i = 0;

						while (i < 100) { // condition pour la boucle
						  i = i + 1;      // corps de la boucle
						}

Boucles - La boucle for

Une boucle for contient à la fois une déclaration de variable, une condition de boucle et une itération


						for (int i = 0; i < 100; i++) {
							//faire quelque chose avec i ici
						}

Boucles - Exercice

  • Parcourir les nombres de 1 à 100
  • Afficher "fizz" si le nombre est divisible par 5
  • Afficher "buzz" si le nombre est divisible par 7
  • Afficher "fizzbuzz" si le nombre est divisible par 5 et par 7
  • Afficher le nombre sinon

Boucles - Exercice


						for(int i = 1; i <= 100; i++) {
						  if (((i % 5) == 0) && ((i % 7) == 0)) {
						    System.out.println("fizzbuzz");
						  } else if ((i % 5) == 0) {
						    System.out.println("fizz");
						  } else if ((i % 7) == 0) {
						    System.out.println("buzz");
						  } else {
						    System.out.println(i);
						  }
						}

Fonctions

Une fonction englobe un morceau de code qui peut ensuite être réutilisé. Une fonction sert surtout à créer une abstraction dans le code.

Exemple - Pythagore


						float a1 = 3.f;
						float b1 = 4.f;
						float c1 = Math.sqrt(a1 * a1 + b1 * b1);

						float a2 = 8.f;
						float b2 = 9.f;
						float c2 = Math.sqrt(a2 * a2 + b2 * b2);

On écrit deux fois la logique du calcul

Exemple - Pythagore(suite)


						float pythagore(float a, float b) {
							return Math.sqrt(a * a + b * b);
						}
						float a1 = 3.f;
						float b1 = 4.f;
						float c1 = pythagore(a1, b1); //Abstraction des calculs

						float a2 = 8.f;
						float b2 = 9.f;
						float c2 = pythagore(a2, b2); //Abstraction des calculs

Listes

Une liste est une collection de variables.

Exemple: Une liste de nombres, une liste de mots, etc.

Listes - Problématique


						string fruit1 = "Banane";
						string fruit2 = "Pomme";
						string fruit3 = "Kiwi";
						string fruit4 = "Poire";
						string fruit5 = "Orange";
						//Etc...

Listes - Solution


					ArrayList<String> fruits = new ArrayList<String>();
					fruits.add("Banane"); //Ajout de "Banane" à la liste.
					fruits.add("Pomme"); //Ajout de "Pomme" à la liste.
					fruits.add("Kiwi"); //Ajout de "Kiwi" à la liste.
					//Etc...

					fruits.get(0); //Retourne "Banane"
					
					fruits.remove("Banane"); //On retire "Banane" de la liste.

					fruits.get(0); //Retourne "Pomme"

Boucler sur une liste

Nous voulons imprimer le contenu de la liste "fruits". 


					//fruits contient ["Pomme", "Kiwi"]
					for(int i = 0; i < fruits.size(); i++) {
						System.out.println(fruits.get(i));
					}

Boucles de jeu

La boucle de jeu permet d'afficher le jeu en continu

Dans cette boucle, nous exécutons toutes les opérations pour passer d'un état à un autre entre chaque affichage

Exemple d'une boucle de jeu


						// Boucle en continu
						while (true) {
						  // Met à jour la position du joueur
						  player.setPosition(Mouse.getPosition());

						  // Dessine le joueur dans la fenêtre
						  player.draw();
						}

À chaque itération, la position du joueur se synchronise avec la position de la souris

Frames

Un frame est une image affichée par la boucle de jeu.

Dans les jeux vidéos, on parle souvent de fps (frame per second)

La norme des jeux modernes est 60fps, donc la boucle effectue 60 itérations par seconde

Frame suite

Comme mentionné dans la boucle d'exécution, un frame effectue des opérations qui changent l'état du jeu

Exemples d'opérations

  • Calcul de la physique
  • Gestion des collisions
  • Calcul de pointages
  • Génération d'obstacles
  • Affichage du jeu

Exemple


						// Boucle en continu
						while (true) {
						  // Calcul de la physique
						  // Gestion des collisions
						  // Calcul de pointage
						  // Génération d'obstacles
						  player.act();

						  // Affichage du jeu
						  player.draw();
						}

Vecteur

Un vecteur est formé de deux composantes, x et y, dans un plan cartésien qui peut indiquer une position ou une direction

Par exemple, dans un jeu, la position du joueur est un vecteur


						// Création d'un vecteur
						Vector2 postion = new Vector2(10, -5);
						// Le vecteur est à la position x = 10, y = -5

Rectangle

Un rectangle est un composé de 4 composantes, x, y, width, height, qui donnent une position et une dimension dans le plan cartésien

Par exemple, un obstacle peut être représenté par un Rectangle dans un jeu


						// Création d'un rectangle
						Rectangle obstacle = new Rectangle(10, -5, 10, 10);
						// Le Rectangle est à la position x = 10, y = -5
						// de dimension width = 10, height = 10

Rectangle - Suite

Plusieurs opérations sont possible avec des Rectangles


						// Création de rectangles
						Rectangle obstacle1 = new Rectangle(10, -5, 10, 10);
						Rectangle obstacle2 = new Rectangle(10, 0, 20, 20);

						// Regarde si les deux rectangles sont superposés
						obstacle1.overlaps(obstacles2);

						// Regarde si obstacle1 contient obstacles2
						obstacle1.contains(obstacles2);

Clock

Une clock agit comme un chronomètre


						// Création de la clock
						Clock clock = new Clock();

						// Obtiens le temps en secondes écoulées depuis le lancement du
						// chronomètre
						clock.getElapsedTime();

						// Relance le chronomètre
						clock.restart();

Sprite Entity

Un sprite entity est une image pouvant être affichée dans la fenêtre de jeu


						// Création du sprite
						SpriteEntity obstacle = new SpriteEntity("obstacle.png");

						// Modification de la position à x = 0, y = 0
						obstacle.setPosition(new Vector2(0, 0));
						// Obtient la position en x
						obstacle.getPosition().x;
						// Obtient la position en y
						obstacle.getPosition().y;
						// Obtient le rectangle de l'entity
						obstacle.getBounds();

						// Afficher le sprite
						obstacle.draw();

Animated Sprite Entity

Un animated sprite entity est une animation 2D pouvant être affichée dans la fenêtre de jeu


						// Création du sprite animé
						AnimatedSpriteEntity player = new AnimatedSpriteEntity("oiseaux.png");

						// Modification de la position à x = 0, y = 0
						player.setPosition(new Vector2(0, 0));

						// Afficher l'animation
						player.draw();
