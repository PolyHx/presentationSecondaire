## Expressions booléennes et conditions Valeurs booléennes possibles: true (vrai) et false (faux) Une expression booléenne est une suite de valeurs booléennes modifiées par des opérateurs logiques (Algèbre de Boole)

Expressions booléennes et conditions - Opérateurs

Comparaison Opérateurs
== (égal) && (et)
!= (non égal) || (ou)
< (plus petit) ! (non)
> (plus grand)
<= (plus petit égal)
>= (plus grand égal)

Boucles

Une boucle while est un morceau de code qui sera répeté tant qu'une condition sera vraie.


						int i = 0;

						while (i < 100) { // condition pour la boucle
						  i = i + 1;      // corps de la boucle
						}

Boucles - La boucle for

Une boucle for contient à la fois une déclaration de variable, une condition de boucle et une itération


						for (int i = 0; i < 100; i++) {
							//faire autre chose ici
						}

Boucles - exercice

  • Parcourir les nombres de 1 à 100
  • Afficher "fizz" si le nombre est divisible par 5
  • Afficher "buzz" si le nombre est divisible par 7
  • Afficher "fizzbuzz" si le nombre est divisible par 5 et par 7
  • Afficher le nombre sinon

Boucles - exercices


						for(int i = 1; i <= 100; i++) {
						  if (((i % 5) == 0) && ((i % 7) == 0))
						    System.out.print("fizzbuzz ");
						  else if ((i % 5) == 0)
						    System.out.print("fizz ");
						  else if ((i % 7) == 0)
						    System.out.print("buzz ");
						  else
						    System.out.print(i);
						}

Fonctions

Une fonction englobe un morceau de code qui peut ensuite être réutilisé. Une fonction sert surtout à créer une abstraction dans le code.

Exemple - Pythagore


						float a1 = 3.f;
						float b1 = 4.f;
						float c1 = Math.sqrt(a1 * a1 + b1 * b1);

						float a2 = 8.f;
						float b2 = 9.f;
						float c2 = Math.sqrt(a2 * a2 + b2 * b2);

On écrit deux fois la logique du calcul

Exemple - Pythagore(suite)


						float pythagore(float a, float b) {
							return Math.sqrt(a * a + b * b);
						}
						float a1 = 3.f;
						float b1 = 4.f;
						float c1 = pythagore(a1, b1); //Abstraction des calculs

						float a2 = 8.f;
						float b2 = 9.f;
						float c2 = pythagore(a2, b2); //Abstraction des calculs

Listes

Une liste est une collection de variables.

Exemple: Une liste de nombres, une liste de mots, etc.

Listes problématique


						string fruit1 = "Banane";
						string fruit2 = "Pomme";
						string fruit3 = "Kiwi";
						string fruit4 = "Poire";
						string fruit5 = "Orange";
						//Etc...

Listes solution


					ArrayList fruits = new ArrayList<>();
					fruits.push("Banane"); //Ajout de "Banane" à la liste.
					fruits.push("Pomme"); //Ajout de "Pomme" à la liste.
					fruits.push("Kiwi"); //Ajout de "Kiwi" à la liste.
					//Etc...

					fruits.get(0); //Retourne "Banane"
					
					fruits.remove("Banane"); //On retire "Banane" de la liste.

					fruits.get(0); //Retourne "Pomme"

Boucler sur une liste

Nous voulons imprimer le contenu de la liste "fruits". 


					//fruits contient ["Pomme", "Kiwi"]
					for(int i = 0; i < fruits.size(); i++) {
						System.out.println(fruits.get(i));
					}
